## 🎯 概要

この設定を行うことで、以下のようになります：

* ✅ PC起動時に自動でNavikoが起動
* ✅ バックグラウンドで待機（画面には表示されない）
* ✅ 「ナビ子」と言うと、画面に表示される
* ✅ 「隠れて」と言うと、再びバックグラウンドに戻る
* ✅ プロセスは常に動作し続ける（音声を監視）

---

## 🔧 前提条件

以下が正常に動作することを確認してください：

1. **Navikoが手動で起動できる**
   ```bash
   python naviko.py
   ```

2. **バックグラウンドモードで起動できる**
   ```bash
   python naviko.py --hidden
   ```
   → ウィンドウが表示されないことを確認

3. **音声コマンドが動作する**
   * 「ナビ子」→ チャットウィンドウが表示される
   * 「隠れて」→ チャットウィンドウが非表示になる

## 📝 設定手順

### ステップ1: スタートアップフォルダを開く

以下のいずれかの方法でスタートアップフォルダを開きます：

#### 方法A: ファイル名を指定して実行

1. **Win + R** キーを押す
2. `shell:startup` と入力
3. **Enter** キーを押す

#### 方法B: エクスプローラーで直接開く

```
C:\Users\<ユーザー名>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
```

**注意**: `<ユーザー名>` は自分のWindowsユーザー名に置き換えてください。

---

### ステップ2: ショートカットを作成

#### 方法A: naviko_launcher.pyw のショートカットを作成（推奨）

1. エクスプローラーでNavikoプロジェクトフォルダを開く
   ```
   例: C:\Users\<ユーザー名>\naviko-lab
   ```

2. `naviko_launcher.pyw` ファイルを見つける

3. **右クリック** → **ショートカットの作成**

4. 作成されたショートカットを **ステップ1で開いたスタートアップフォルダ** にドラッグ＆ドロップ

5. （オプション）ショートカット名を変更
   * 右クリック → **名前の変更**
   * 例: `Naviko自動起動`

#### 方法B: 手動でショートカットを作成

スタートアップフォルダ内で：

1. **右クリック** → **新規作成** → **ショートカット**

2. **項目の場所** に以下を入力（パスは環境に合わせて変更）：
   ```
   C:\Users\<ユーザー名>\AppData\Local\Programs\Python\Python311\pythonw.exe "C:\Users\<ユーザー名>\naviko-lab\naviko_launcher.pyw"
   ```
   
   **重要**:
   * `pythonw.exe` を使用（`python.exe` ではなく）
   * `.pyw` ファイルを指定

3. **次へ** をクリック

4. ショートカット名を入力（例: `Naviko自動起動`）

5. **完了** をクリック

---

### ステップ3: ショートカットのプロパティ確認（オプション）

作成したショートカットを右クリック → **プロパティ** で以下を確認：

* **リンク先**:
  ```
  "C:\...\pythonw.exe" "C:\...\naviko_launcher.pyw"
  ```

* **作業フォルダー**:
  ```
  C:\Users\<ユーザー名>\naviko-lab
  ```
  （空欄の場合は、Navikoプロジェクトフォルダのパスを入力）

* **実行時の大きさ**: `通常のウィンドウ` または `最小化`

## ✅ 動作確認

### テスト1: 手動実行

1. スタートアップフォルダ内のショートカットをダブルクリック

2. 以下を確認：
   * コンソールウィンドウが表示されないこと
   * Navikoプロセスが起動していること
     * タスクマネージャー（Ctrl+Shift+Esc）→ 詳細 → `python.exe` または `pythonw.exe`

3. 音声コマンドをテスト：
   * 「ナビ子」→ チャットウィンドウが表示される
   * 「隠れて」→ チャットウィンドウが非表示になる

### テスト2: PC再起動で自動起動確認

1. PCを再起動

2. ログイン後、数秒待つ

3. タスクマネージャーで `python.exe` または `pythonw.exe` プロセスが起動しているか確認

4. 「ナビ子」と言ってチャットウィンドウが表示されるか確認

## 🔍 トラブルシューティング

### 問題1: PC起動後もNavikoが起動しない

**原因と対策**:

1. **ショートカットのパスが間違っている**
   * ショートカットのプロパティを確認
   * `naviko_launcher.pyw` のパスが正しいか確認

2. **Pythonのパスが間違っている**
   * コマンドプロンプトで `where python` を実行
   * 正しいPythonのパスをショートカットに設定

3. **作業フォルダーが未設定**
   * ショートカットのプロパティ → **作業フォルダー** にNavikoプロジェクトフォルダのパスを設定

4. **エラーログを確認**
   * Navikoプロジェクトフォルダ内の `naviko_launcher_error.log` を確認
   * エラーメッセージから原因を特定

### 問題2: コンソールウィンドウが表示される

**原因と対策**:

1. **pythonw.exe を使用していない**
   * ショートカットのリンク先を `pythonw.exe` に変更
   * または `.pyw` 拡張子のファイルを使用

2. **ショートカットで python.exe を指定している**
   * `python.exe` → `pythonw.exe` に変更

### 問題3: 音声コマンドが動作しない

**原因と対策**:

1. **マイクが認識されていない**
   * Windowsの設定 → サウンド → 入力デバイスを確認

2. **Voskモデルが見つからない**
   * `C:\vosk_models\vosk-model-small-ja-0.22` が存在するか確認

3. **音声認識の設定を確認**
   * 手動でNavikoを起動（`python naviko.py --hidden`）
   * コンソールに音声認識のログが表示されるか確認

### 問題4: 起動ログに記録されない

**原因と対策**:

1. **ランチャーが正常に実行されていない**
   * 手動でショートカットをダブルクリックしてテスト
   * エラーが発生する場合は `naviko_launcher_error.log` を確認

2. **ログファイルへの書き込み権限がない**
   * Navikoプロジェクトフォルダへの書き込み権限を確認

## 🚫 自動起動の無効化

自動起動を停止したい場合：

### 一時的に無効化

1. スタートアップフォルダを開く（**Win + R** → `shell:startup`）

2. Navikoのショートカットを **別の場所に移動**（削除しない）

3. PCを再起動

### 完全に削除

1. スタートアップフォルダを開く（**Win + R** → `shell:startup`）

2. Navikoのショートカットを **削除**

3. PCを再起動
