## 📊 重要な技術的知見

### 3方向同期の安全な手順
1. Workspace側の変更をバックアップ
2. `git reset --hard origin/main`で強制同期
3. 最新版に既に実装済みの内容が含まれているか確認

### Groq API 401エラー診断フロー
```
1. 環境変数確認
   ↓
2. APIテストスクリプト実行
   ↓
3. ステータスコード確認
   - 200: 成功
   - 401: APIキー無効 → Groqダッシュボード確認
   ↓
4. 新しいAPIキー生成
   ↓
5. 環境変数再設定
```
